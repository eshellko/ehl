Введение

В данном документе приводятся технические характеристики, функционал и инструкции по применению блока ehl_m1553, имеющего следующие основные зхарактеристики:

  • работа в режиме Контроллера Шины (КШ) / Bus Controller (BC);
  • работа в режиме Оконечное Устройство (ОУ) / Remote Terminal (RT);
  • работа в режиме Монитор Шины (МШ) / Bus Monitor (BM);
  • поддержка резервной магистральной шины;
  • поддерживаемые типы передач (включая широковещательные): BC->RT, RT->BC, RT->RT;
  • поддержка до 31 RT;
  • проверка соответствия протоколу;
  • сохранение статистики шины.
