Регистры статистики передатчика

TX_STAT_TFTO

Биты Название Описание Доступ Начальное значение
31:0 - Содержит количество удачно переданных кадров. R x

TX_STAT_TSCF

Биты Название Описание Доступ Начальное значение
31:0 - Содержит количество удачно переданных кадров, при передаче которых возникла ровно 1 коллизия и переданных со второй попытки. R x

TX_STAT_TMCF

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, в процессе передачи которых возникло больше 1 коллизии, и переданных успешно. R x

TX_STAT_TOTO0: Младшие 32 бита.

TX_STAT_TOTO1: Старшие 32 бита.

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик байт данных и PAD в успешно переданных кадрах. Регистр CPB_TOTO0 содержит младшие 32 бита, CPB_TOTO1 — старшие 32 бита. R x

TX_STAT_TFWDX

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, передача которых была отложена при первой попытке из-за того, что среда передачи была занята. Не учитываются кадры, при передаче которых возникла хотя бы одна коллизия. R x

TX_STAT_TLC

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, в процессе передачи которых возникала поздняя коллизия. Поздней считается коллизия, обнаруженная после передачи 512-го бита передаваемого кадра. R x

TX_STAT_TFADTXS

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, передача которых прекращена из-за превышения допустимого числа коллизий. R x

TX_STAT_TFLDTMXE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, не переданных из-за ошибок заполнения FIFO. R x

TX_STAT_TCSE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, при передаче которых не устанавливался сигнал CRS или CRS сбросился до окончания передачи. Этот счетчик инкрементируется только в режиме Half Duplex. R x

TX_STAT_TMFXO

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик успешно переданных кадров, в поле Destination Address которых находятся групповой не широковещательный адрес. R x

TX_STAT_TBFXO

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик успешно переданных кадров, в поле Destination Address которых находится широковещательный адрес. R x

TX_STAT_TFWED

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, передача которых была отложена на длительный период времени (более 24288 битовых времен). Счетчик инкрементируется 1 раз для кадра, при передаче которого возникло длительное ожидание, независимо от количества попыток передать кадр и количества возникших задержек передачи. R x

Биты Название Описание Доступ Начальное значение
31:0 - Гистограмма активности коллизий. Индекс каждого регистра соответствует числу коллизий, произошедших при передаче кадра. Каждый элемент массива содержит счетчик успешно переданных кадров, при передаче которых возникло соответствующее число коллизий. Счетчики инкрементируются только в режиме половинного дуплекса. R x
