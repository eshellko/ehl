Параметры

ehl_emac имеет возможность быть настроенным под задачу пользователя. За настройку отвечают параметры, представленные далее.

Название Значение по умолчанию Допустимые значения Описание
CDC_SYNC_STAGE 3 0-3 Количество триггеров в синхронизаторах контроллера.
DATA_WIDTH 32 16, 32, 64, 128, 256 Ширина шины данных.
TX_DESCR_P0_COUNT 8 0-32 Количество дескрипторов передатчика с приоритетом 0 (высший)
TX_DESCR_P1_COUNT 8 0-32 Количество дескрипторов передатчика с приоритетом 1
TX_DESCR_P2_COUNT 8 0-32 Количество дескрипторов передатчика с приоритетом 2
TX_DESCR_P3_COUNT 8 0-32 Количество дескрипторов передатчика с приоритетом 3 (низший)
RX_DESCR_COUNT 32 2-32 Количество дескрипторов приемника. Если RX_ENA равен 0, то значение этого параметра не важно.
TX_ENA 1 0, 1 Наличие передатчика
TX_CPB_ENA 1 0, 1 Наличие регистров CAPABILITY передатчика. Может быть равен 1 только если TX_ENA равен 1.
RX_ENA 1 0, 1 Наличие приемника
RX_CPB_ENA 1 0, 1 Наличие регистров CAPABILITY приемника. Может быть равен 1 только если RX_ENA равен 1.
MDIO_ENA 1 0, 1 Наличие контроллера MIIM
FC_ENA 1 0, 1 Наличие аппаратной поддержки Flow Control. Может быть равен 1 только если RX_ENA и TX_ENA оба равны 1.
SYNCHRONIZE_RESETS 0 0, 1 Наличие внутренних синхронизаторов сброса.
DEFAULT_MAC 0x000000000000 Любое значение, бит 40 всегда равен 0. MAC адрес устройства по умолчанию.
