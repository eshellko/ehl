Регистры статистики приемника

RX_STAT_RFTLE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик принятых кадров, размер которых превышает заданный регистром MAXMIN.maxlr. R x

RX_STAT_RORO0: Младшие 32 бита.

RX_STAT_RORO1: Старшие 32 бита.

Биты Название Описание Доступ Начальное значение
63:0 - Счетчик байт данных (включая пад) в успешно принятых кадрах. Счетчик не инкрементируется, если кадр содержит ошибки размера, контрольной суммы, смещения или переполнения буфера. R x

RX_STAT_RFRO

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик успешно принятых кадров. R x

RX_STAT_RFCSE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик принятых кадров, имеющих целое количество байт и ошибку контрольной суммы. R x

RX_STAT_RMFRO

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик принятых кадров, имеющих в поле адреса места назначения групповой адрес (регистр MULT или 0x0180C2000001). Если кадр содержит ошибки, счетчик не инкрементируется. R x

RX_STAT_RBFRO

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик принятых кадров, имеющих 0xFFFFFFFFFFFF в поле адреса места назначения. Если кадр содержит ошибки, счетчик не инкрементируется. R x

RX_STAT_ROORLF

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик принятых кадров со значением в поле Length/Type, превышающем 0x5DC, но менее 0x600. R x

RX_STAT_RIRLE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров со значением в поле Length/Type, меньшим либо равным 0x5DC, но не соответствующем количеству принятых байт. Данный счетчик учитывает кадры, дополненные полем PAD. R x

RX_STAT_RAE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, состоящих из нецелого числа байт и имеющих ошибку контрольной суммы. R x

RX_STAT_RFLDTIMRE

Биты Название Описание Доступ Начальное значение
31:0 - Счетчик кадров, не принятых из-за внутренних ошибок приемника (переполнение буфера приемника). Счетчик инкрементируется, если кадр не содержит других ошибок. R x
