2. Соответствие модели утверждениям из спецификации

Данная проверка подразумевает верификацию соответствия RTL модели спецификации контроллера. Каждое утверждение имеет ID, отражаемый в процессе моделирования. ID служит для построения соответствия между спецификацией контроллера, спецификацией тестирования и тестом.

ID Утверждение Тип проверки
RST-1-0 Сброс работающего (считает, прерывания разрешены) таймера может привести (и приводит) к генерации прерывания при сброса таймера. Мониторинг появления прерывания при выполнении условий теста.

3.1. Проверка свойств контроллера

Тест Описание Проверяемые свойства
ONESHOT Таймер загружается различными значениями и запускается на счет в режиме "Таймер".
  • Остановка по окончании счета в однократном режиме.
  • Перезапуск по окончании интервала в циклическом режиме.
  • Длительность интервала счета.
