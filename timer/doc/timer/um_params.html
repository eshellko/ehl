Параметры конфигурации

ehl_timer имеет возможность быть настроенным под задачу пользователя. За настройку отвечают параметры, представленные далее.

Название Значение по умолчанию Допустимые значения Описание
TECHNOLOGY 0 0-...* Выбор технологического процесса для технологически специфичных элементов дизайна.
CDC_SYNC_STAGE 2 0-3 Количество последовательно включенных триггеров в синхронизаторах контроллера:
  • 0 - комбинаторный путь между данными (допускается только при использовании общего синхросигнала для тактирования управляющих регистров и таймеров)
  • 1, 2, 3 - заданное количество триггеров
    		•
    BUS_WIDTH 32 8, 32 Разрядность системной шины данных.
    NTIMERS 1 1-4 Количество независимых таймеров в составе контроллера. Позволяет использовать одно устройство на системной шине для нескольких таймеров.
    TIMER_WIDTH 32 8, 16, 24, 32 Разрядность таймера.
    SYNC_MODE 0 0, 1 Синхронный режим:
  • 0 - clk и tmr_clk асинхронны, переходы данных между ними защищены синхронизаторами;
  • 1 - clk и tmr_clk тактируются одним источником.
    		•
    PWM_ENA 0 0-15 Разрешение использования функционала 3-х канального ШИМ для выбранного таймера. 1 в соответствующем разряде разрешает ШИМ для выбранного таймера.
  • [0]: 1 - ШИМ для таймера 0 реализован; 0 - таймер 0 не содержит ШИМ.
  • [1]: 1 - ШИМ для таймера 1 реализован; 0 - таймер 1 не содержит ШИМ.
  • [2]: 1 - ШИМ для таймера 2 реализован; 0 - таймер 2 не содержит ШИМ.
  • [3]: 1 - ШИМ для таймера 3 реализован; 0 - таймер 3 не содержит ШИМ.
    		•
    CPT_ENA 0 0-15 Разрешение использования функционала "захват" / "счетчик" для выбранного таймера. 1 в соответствующем разряде разрешает захват для выбранного таймера.
  • [0]: 1 - захват для таймера 0 реализован; 0 - таймер 0 не содержит функционал захвата / счетчика.
  • [1]: 1 - захват для таймера 1 реализован; 0 - таймер 1 не содержит функционал захвата / счетчика.
  • [2]: 1 - захват для таймера 2 реализован; 0 - таймер 2 не содержит функционал захвата / счетчика.
  • [3]: 1 - захват для таймера 3 реализован; 0 - таймер 3 не содержит функционал захвата / счетчика.
    		•
    CMP_ENA 0 0-15 Разрешение использования функционала "сравнение" для выбранного таймера. 1 в соответствующем разряде разрешает сравнение для выбранного таймера.
  • [0]: 1 - сравнение для таймера 0 реализовано; 0 - таймер 0 не содержит функционал сравнения.
  • [1]: 1 - сравнение для таймера 1 реализовано; 0 - таймер 1 не содержит функционал сравнения.
  • [2]: 1 - сравнение для таймера 2 реализовано; 0 - таймер 2 не содержит функционал сравнения.
  • [3]: 1 - сравнение для таймера 3 реализовано; 0 - таймер 3 не содержит функционал сравнения.
    		•
    CPT_SYNC 255 0-255 Управление наличием (1) или отсутствием (0) синхронизатора на соответствующем входе cpt_in. Младший бит параметра соответствует младшему бита входа.
    USE_CG 1 0, 1 Разрешение использования Clock Gating элементов в контроллере.
  • 1 - применять clock gating и регистры TMRx_PRE;
  • 0 - не использовать clock gating для технологий не поддерживающих указанные элементы.
    		•
