Введение

В данном документе приводятся технические характеристики, функционал и инструкции по применению блока ehl_timer, имеющего следующие основные характеристики:

  • Конфигурируемое количество независимых таймеров (1-4);
  • Конфигурируемая разрядность таймера (8, 16, 24, 32 бит).;
  • Конфигурируемая ширина шины управления (8, 32 бит);
  • Конфигурируемый ШИМ (до 3 каналов) для каждого таймера;
  • Конфигурируемый режим сравнения (3 канала) для каждого таймера (2 регистра сравнения);
  • Конфигурируемый режим захвата для каждого таймера (программный выбор одного из 2 внешних каналов захвата);
  • Таймер в режиме однократного и циклического счета с возможностью остановки;
  • Программное управление направлением счета (вверх, вниз).
  • Программируемый коэффициент деления частоты таймера: 1 - 256.
