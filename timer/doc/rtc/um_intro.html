Введение

В данном документе приводятся технические характеристики, функционал и инструкции по применению блока ehl_rtc, имеющего следующие основные зхарактеристики:

  • часы реального времени с шагом 0,1 с;
  • календарь с коррекцией даты для високосного года;
  • режим будильника с генерацией прерывания при совпадении даты / времени;
