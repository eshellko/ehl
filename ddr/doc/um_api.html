Базовые функции

API_DDR_INIT: инициализация контроллера памяти

int API_DDR_INIT(unsigned int cnum, struct ddr_ctrl dc[]);
   cnum       - количество контроллеров памяти
   dc         - массив структур параметров контроллеров памяти

Возвращаемые значения

DDR_INIT_RV_OKAY        - инициализация завершена успешно.
DDR_INIT_RV_TRNG_FAILED - ошибка в процессе тренинга.
DDR_INIT_BAD_CONFIG     - некорректная настройка параметров контроллера.
DDR_INIT_TIMEOUT_EXCEED - ошибка, время ожидания операции превышено. Контроллер может оказаться в нерабочем состоярнии, выполняя последнюю операцию.
