Параметры конфигурации

ehl_lssi имеет возможность быть настроенным под задачу пользователя. За настройку отвечают параметры, представленные далее.

Название Разрядность Значение по умолчанию Допустимые значения Описание
FIFO_SIZE 16 16 4, 8, 16, 32, 64, 128 Количество байт в буферах fifo_tx и fifo_tx.
TICKS_TO_BIT 2 0 0-2 Определяет число стробов RCLK (выход частоты делителя синхросигнала) на 1 бит принимаемых/передаваемых данных в режиме UART. Увеличение этого параметра приводит к небольшому росту аппаратуры, но позволяет получить более точную синхронизацию с устройством на шине UART. Может принимать следующие значения:
  • 0 - бит данных стробируется 4 тактами RCLK;
  • 1 - бит данных стробируется 8 тактами RCLK;
  • 2 - бит данных стробируется 16 тактами RCLK.
    		•
    SS_WIDTH 32 8 1-8 Количество бит в регистре SSR - количество управляемых контроллером сигналов SPI Slave Select.
    ECC_ENA 1 0 0, 1 Применение Single Error Correction логики в FIFO приемо-передатчиков. 1 - добавляет SEC в буферы при записи и исправляет (без сигнализации) при чтении.
    TECHNOLOGY 32 0 см. techmap/doc/readme* Технологический базис для отдельных элементов дизайна - CDC синхронизаторов, clock gating.
    USE_CG 1 1 0, 1 Разрешение использования Clock Gating элементов в контроллере.
  • 1 - применять clock gating и регистр CDR;
  • 0 - не использовать clock gating для технологий не поддерживающих указанные элементы.
    		•
    DEVICE_CFG 2 3 1-3 Реализация логики работы схемы. Бит 1 отвечает за UART, бит 0 за SPI. 1 в разряде реализует логику в контроллере.
